Asignacion 12: Árboles Binarios de Búsqueda

#Caracteristicas: El siguiente proyecto contiene la implementacion completa de un Árbol Binario de Búsqueda o tambien llamado(BST). Nuestro objetivo principal es comprender su estructura para aplicar las operaciones fundamentales que conlleva, ademas de analizar cual es el funcionamiento de los distintos tipos de recorridos.

Funcion 1: Primero implementamos desde cero un Arbol Binario de Busqueda. Funcion 2: Dominar las operaciones principales del BST: las cuales son insercion, busqueda y eliminacion. Funcion 3: Comprender y aplicar los diferentes recorridos que maneja el arbol (preorden, inorden y postorden). Funcion 4: Analizar como es que un BST puede aplicarse para resolver problemas del mundo real. Funcion 5: Utilizar Git/GitHub para documentar y poder gestionar el desarrollo del proyecto.

#Ejercicios realizados
Implementación del Nodo y del Arbol BST. Insercion de los nodos: Agregar los elementos dentro del BST respetando sus reglas de orden. Busqueda de valores: Localizar elementos dentro del arbol de una manera eficiente. Eliminacion de nodos: Eliminar elementos considerando los distintos casos como lo son...(nodo hoja, puede ser con un hijo o incluso con dos hijos). Recorridos del arbol: Preorden, Inorden, Postorden Aplicacion del BST: Resolver un problema de forma practica utilizando el arbol implementado.
